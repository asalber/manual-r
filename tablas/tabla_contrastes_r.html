Contrastes de hipótesis con R

Test Sintaxis (parámetros comunes) Ejemplo
t de Student (1 muestra) t.test(x, mu = 0, alternative = c("two.sided","less","greater"), conf.level = 0.95) t.test(x, mu = 5)
t de Welch (2 muestras independientes) t.test(y ~ g, var.equal = FALSE, alternative = "two.sided", conf.level = 0.95) t.test(Sepal.Length ~ Species, data = iris[iris$Species!="setosa",])
t pareada (2 medidas relacionadas) t.test(x, y, paired = TRUE, alternative = "two.sided", conf.level = 0.95) t.test(pre, post, paired = TRUE)
ANOVA 1 factor aov(y ~ g, data = df) fit <- aov(Sepal.Length ~ Species, data = iris)
ANOVA 2 factores (+ interacción) aov(y ~ A * B, data = df) aov(y ~ grupo * tratamiento, data = df)
ANCOVA lm(y ~ grupo + covar, data = df) (contraste vía anova(fit)) lm(y ~ grupo + edad, data = df)
Post-hoc (Tukey) emmeans::emmeans(fit, "g") | > emmeans::pairs(adjust = "tukey") fit <- aov(y ~ g, df); emmeans::emmeans(fit,"g") | > emmeans::pairs("tukey")
Correlación de Pearson cor.test(x, y, method = "pearson", alternative = "two.sided", conf.level = 0.95) cor.test(iris$Sepal.Length, iris$Sepal.Width)
Correlación de Spearman cor.test(x, y, method = "spearman") cor.test(x, y, method = "spearman")
Regresión lineal lm(y ~ x + z, data = df); resumen: summary(fit) fit <- lm(mpg ~ wt + hp, data = mtcars)
Mann–Whitney (Wilcoxon 2 indep.) wilcox.test(y ~ g, alternative = "two.sided", conf.level = 0.95, exact = FALSE) wilcox.test(y ~ g, data = df)
Wilcoxon pareado (signed-rank) wilcox.test(x, y, paired = TRUE, alternative = "two.sided", exact = FALSE) wilcox.test(pre, post, paired = TRUE)
Kruskal–Wallis (≥3 indep.) kruskal.test(y ~ g, data = df) kruskal.test(Sepal.Length ~ Species, data = iris)
Friedman (≥3 relacionados, RM) friedman.test(y ~ within | id, data = df) friedman.test(valor ~ tiempo | sujeto, data = df)
Igualdad de varianzas (Levene) car::leveneTest(y ~ g, data = df, center = median) car::leveneTest(mpg ~ cyl, data = mtcars)
F test de varianzas (2 grupos) var.test(x, y, alternative = "two.sided", conf.level = 0.95) var.test(x, y)
Normalidad (Shapiro–Wilk) shapiro.test(x) shapiro.test(rnorm(50))
Normalidad (Anderson–Darling) nortest::ad.test(x) nortest::ad.test(x)
Normalidad (D’Agostino–Pearson) fBasics::dagoTest(x) fBasics::dagoTest(x)
χ² de independencia (r×c) chisq.test(tabla) o chisq.test(x = df$A, y = df$B) chisq.test(table(df$sexo, df$fuma))
Fisher exacta (2×2 o celdas pequeñas) fisher.test(tabla, alternative = "two.sided") fisher.test(matrix(c(5,1,2,6),2))
Proporción 1 muestra (exacta) binom.test(x = éxitos, n = n, p = p0, alternative = "two.sided", conf.level = 0.95) binom.test(42, 100, p = 0.5)
Proporciones 1 o 2 muestras (aprox. z) prop.test(x = c(s1,s2), n = c(n1,n2), correct = TRUE); 1 muestra: prop.test(s, n, p = p0) prop.test(c(30,45), c(100,110))
McNemar (binaria pareada) mcnemar.test(tabla_2x2) mcnemar.test(matrix(c(10,3,6,21),2))
